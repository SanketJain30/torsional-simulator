// Theory — Torsional Vibration of Two Rotor System

What is Torsional Vibration?

Torsional vibration occurs when a shaft experiences oscillatory twisting about its longitudinal axis. In a two-rotor system, two discs (rotors) are connected by an elastic shaft. When one disc is given an angular displacement and released, both discs oscillate about the shaft axis.

Natural Frequency Formula

The non-trivial (non-zero) natural frequency of the two-rotor system is derived from the equations of motion:

// DERIVATION
I₁·θ̈₁ + K·(θ₁ − θ₂) = 0    ... (1)
I₂·θ̈₂ − K·(θ₁ − θ₂) = 0    ... (2)

Assuming θ = A·sin(ωt), solving eigenvalue problem:

ωₙ = √[ K·(I₁ + I₂) / (I₁·I₂) ]

Node Point

The node is a point on the shaft that remains stationary during vibration. Its position from rotor 1 is:

// NODE LOCATION
l₁ = L × I₂ / (I₁ + I₂)
l₂ = L × I₁ / (I₁ + I₂)
where L = total shaft length

Mode shape ratio: θ₁/θ₂ = −I₂/I₁    (opposite phase)

Experimental Method

Measure time for N oscillations using a stopwatch. The experimental natural frequency is:
fₑₓₚ = N / t  [Hz]  |  ωₑₓₚ = 2π × fₑₓₚ  [rad/s]

Percentage Error

% Error = |ωₙ_analytical − ωₑₓₚ| / ωₙ_analytical × 100
Acceptable error in lab: typically ≤ 5–10%. Causes: shaft imperfections, friction, measurement delay.
